# Планировщик заявок технического отдела

Статический мини-сайт для ООО «ЭРА Глонасс»: приветственная главная и план загрузки инженеров по часам и дням месяца.

## Состав

- `index.html` — лендинг и ссылки в планировщик.
- `planner.html` — форма заявок, таблица расписания, архив месяцев, сводка по инженерам, печать.

Данные планов хранятся в браузере (`localStorage`, ключ `techDeptMonthlyPlansV1`).

## Запуск

1. **Основной способ:** откройте `index.html` в браузере (двойной щелчок или перетаскивание в окно браузера).
2. **Опционально (нужен [Node.js](https://nodejs.org/)):** локальный сервер — удобно смотреть с телефона в одной Wi‑Fi сети.
   ```bash
   npm start
   ```
   В терминале появится URL (порт задаётся в `package.json`, по умолчанию `3000`).

Для `npm start` нужен установленный [Node.js](https://nodejs.org/). Сам сайт на чистом HTML/CSS/JS и сервер не обязателен.
