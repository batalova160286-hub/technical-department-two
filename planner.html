Документация

Сайт статический: откройте index.html для главной страницы или этот файл (planner.html) для планировщика. Работает без сервера.

Рабочее время технического отдела: с 9:00 до 20:00. Каждый час представлен отдельной ячейкой.

Дни в таблице визуально разделены горизонтальными границами, чтобы удобнее читать расписание по датам.

Обеденные перерывы:
Третьяков А.В. — 12:00–13:00
Бобылев С.Б. — 13:00–14:00
Долгополов Д.Н. — 14:00–15:00

Для добавления заявки выберите дату, инженера, время, тип заявки, длительность и нажмите Добавить заявку.

В графике всегда отображаются все дни месяца: и рабочие, и выходные.

Таблица дополнительно разделяется по неделям отдельными строками, чтобы в длинном месяце было проще ориентироваться при прокрутке вниз.

В блоке «Архив месяцев» можно сохранить текущий месяц и в любой момент открыть ранее сохраненный план.

Если заявка пересекается с обедом, обед автоматически переносится на ближайший свободный час и отмечается светящейся лампочкой. Конфликт с уже назначенной заявкой по-прежнему блокируется.

Для удаления конкретной заявки нажмите на занятую ячейку в таблице.

Кнопка Печать формирует печатную версию плана.

Действие с заявкой

Выберите, что нужно сделать с выбранной заявкой.
