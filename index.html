Добро пожаловать

Внутренний инструмент для распределения заявок по инженерам, дням и часам. Всё хранится в браузере — можно работать офлайн после первой загрузки страницы.

Открыть план загрузки Возможности
📅

Месячный график

Все дни месяца, разбивка по неделям и трём инженерам с 9:00 до 20:00.

💾

Архив месяцев

Сохранение и открытие планов в localStorage — без отдельной базы.

🍽️

Обеды

Учёт личных слотов; при конфликте с заявкой обед сдвигается на свободный час.

🖨️

Печать

Кнопка печати убирает лишнее и оставляет чистый план на бумаге.

Как открыть: достаточно двойного щелчка по index.html. Если установлен Node.js, в папке проекта можно выполнить npm start — тогда сайт откроется через локальный сервер (удобно для просмотра с телефона в той же сети).
